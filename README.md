Department Management System (DMS) is a web application designed to reduce paperwork and help manage academic operations digitally for a University department. Typically, Students have to fill out numerous UG forms, such as Transcript Request, FYP Proposals, Leave applications, on paper, which often get misplaced, take a lot of time for approvals, and aren't transparent. Thus, DMS allows Students, Teachers, and Administrators to have their own role-based portal for performing required operations.Being a web app, it can be deployed for multiple departments and even multiple institutions. On testing, DMS obtained successful testing without errors, covering all the test cases, which concludes our effort towards providing a hassle-free solution to department paperwork.
